# About

Repository contains three tests for website https://www.sapfioneer.com/

## Installation

- clone repository
- type following commands in the project folder:

```bash
npm install cypress@latest
```
```bash
npm install --save-dev typescript
```
```bash
npx cypress open
```
or
```bash
npx cypress run
```
